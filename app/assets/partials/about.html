sapling-aego

Sapling Backend Written in Go - Engineered to Run on Google App Engine

How to use sapling-aego

  1. git clone https://github.com/scotch/sapling-aego.git to clone the sapling-aego repository
  2. cd sapling-aego
  3. git submodule init
  4. git submodule update
  5. cd client/
  6. ./scripts/init.sh to install node packages
  7. ./scripts/development.sh to compile javascripts and css

Using Jade?

  1. ./scripts/compile-jade.sh to run a jade watcher

Or Using HTML?

  1. ./scripts/compile-html.sh to compile html files in app/assets
